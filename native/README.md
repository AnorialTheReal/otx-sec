# Native Modules

OTX-Sec is Linux-first.

Python handles:
- GUI
- orchestration
- threat intelligence APIs
- reporting
- service logic

Native modules are planned for performance-critical work.

## Rust

Rust is the first target for:
- fast SHA256 hashing
- entropy calculation
- ELF detection
- low-memory file scanning

## C / C++

Reserved for future Linux low-level helpers.

## Go

Reserved for future process and network helper tools.

## C#

Reserved for optional future tooling.
Not used for the current Linux-first antivirus scope.
