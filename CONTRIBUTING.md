# Contributing to OTX-Sec

Thank you for considering contributing to OTX-Sec.

## How to Contribute

### Reporting Bugs

Please include:

* Linux Distribution
* OTX-Sec Version
* Python Version
* Steps to reproduce
* Logs if available

### Feature Requests

Please explain:

* Problem
* Proposed Solution
* Expected Benefit

### Pull Requests

Requirements:

* Follow project coding standards
* Test your code
* Keep changes focused
* Update documentation if needed

## Development Setup

```bash
git clone https://github.com/AnorialTheReal/otx-sec.git
cd otx-sec

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt
```

## Code Style

* Clear variable names
* Comments where necessary
* Avoid hardcoded paths
* Prefer configuration files

## Security

Never commit:

* API Keys
* Tokens
* Passwords
* Personal Data

Thank you for helping improve OTX-Sec.
